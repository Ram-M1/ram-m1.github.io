Политика конфиденциальности

Настоящая Политика описывает, как приложение FOCUS («мы») обрабатывает персональные данные пользователей в соответствии с Федеральным законом РФ №152-ФЗ «О персональных данных».

1. Какие данные мы собираем

  • Имя (или псевдоним), адрес электронной почты, номер телефона;
  • Дата рождения, пол, город;
  • Данные, которые вы вносите сами: питание, тренировки, сон, настроение, привычки, цели и другие записи для самоотслеживания;
  • Данные о состоянии здоровья (сведения о приёме лекарств, БАДов, физических тренировках) — относятся к специальной категории персональных данных. Вы вносите их добровольно, и мы обрабатываем их только на основании вашего отдельного согласия, выраженного при регистрации, исключительно для работы функций самоотслеживания и напоминаний;
  • Технические данные, необходимые для работы приложения.

2. Зачем мы собираем данные

  • Для работы вашего личного кабинета и функций приложения;
  • Для сохранения ваших записей и синхронизации между устройствами;
  • Для связи между пользователями (коуч и клиент, консультации);
  • Для улучшения работы сервиса.

3. Как мы защищаем данные

Пароли не хранятся у нас в открытом виде — они защищены средствами сервиса авторизации. Передача данных происходит по защищённому соединению (HTTPS). Доступ к вашим данным имеете только вы; другие пользователи не видят ваши личные записи.

4. Передача данных третьим лицам

Мы не продаём и не передаём ваши персональные данные третьим лицам для рекламы. Данные могут обрабатываться нашими техническими партнёрами (облачные сервисы) исключительно для обеспечения работы приложения.

5. Ваши права

Вы можете в любой момент: получить информацию о своих данных, изменить их, а также удалить свой аккаунт и все данные в настройках приложения. После удаления данные стираются без возможности восстановления.

6. Согласие

Регистрируясь в приложении, вы даёте согласие на обработку своих персональных данных на условиях настоящей Политики. Согласие можно отозвать, удалив аккаунт.

7. Контакты

По вопросам обработки данных пишите нам на почту, указанную в разделе «О приложении».
